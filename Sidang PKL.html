Pendahuluan

Latar belakang, tujuan, dan manfaat pelaksanaan PKL di PT Metanouva Informatika

Latar Belakang

PKL adalah kegiatan wajib bagi siswa SMK untuk menerapkan ilmu yang dipelajari di sekolah ke dunia kerja nyata. Bagi siswa Jurusan RPL di SMK Negeri 1 Manggar, PKL ini penting untuk:

  • Mengasah keterampilan pemrograman
  • Memahami cara kerja di perusahaan TI
  • Memenuhi syarat mengikuti Ujian Kompetensi Keahlian (UKK)

Selama PKL, saya bertanggung jawab memimpin, menginspirasi, dan mengarahkan tim pengembangan aplikasi POS Kasir dengan tugas:

  • Pembagian tugas sesuai minat dan kemampuan
  • Membuat alur aplikasi dan konsep (Flowchart)
  • Membuat Dokumen Teknis dan Manual Book aplikasi
  • Menyiapkan materi presentasi

Tujuan PKL

  • Menerapkan konsep dan keterampilan pemrograman web (PHP, OOP, MVC) ke dalam proyek nyata
  • Menguasai alat dan teknologi industri (Git/GitHub, Laravel)
  • Mengembangkan modul Admin Panel lengkap untuk aplikasi POS Kasir
  • Memahami dan mengimplementasikan sistem keamanan dasar (Role-Based Access Control)
  • Melatih kemampuan kolaborasi dalam tim dengan pembagian tugas terstruktur
  • Membentuk sikap profesional, adaptasi, kerja sama tim, dan disiplin

Manfaat PKL

  • Bagi Siswa: Pengalaman nyata pengembangan software, peningkatan skill pemrograman, belajar kerja sama tim dan komunikasi
  • Bagi Sekolah: Evaluasi kurikulum, hubungan dengan industri
  • Bagi Perusahaan: Rekrutmen potensial, kontribusi pendidikan

Landasan Teori

Konsep dan teknologi yang digunakan dalam pengembangan aplikasi POS Kasir

Website & Sistem POS

Website: Sekumpulan halaman digital yang saling terhubung, berisi teks, gambar, video, dan elemen interaktif lainnya yang dapat diakses melalui internet.

Sistem POS (Point of Sale): Sistem terkomputerisasi yang menggabungkan hardware dan software untuk memproses transaksi penjualan, mengelola inventaris, pelanggan, dan laporan bisnis secara otomatis.

Git & GitHub

Git: Sistem kontrol versi terdistribusi untuk melacak perubahan kode secara lokal.

GitHub: Platform berbasis cloud yang menyediakan hosting repositori Git, kolaborasi, dan manajemen proyek.

Digunakan untuk kolaborasi tim dalam pengembangan aplikasi.

Laravel Framework

Framework PHP open-source modern yang menerapkan pola arsitektur Model-View-Controller (MVC) untuk struktur kode yang terorganisir.

PHP 8.3+ MVC Pattern Eloquent ORM Blade Template

OOP (Object-Oriented Programming)

Paradigma pemrograman yang menjadikan objek sebagai unit utama, merepresentasikan entitas dunia nyata dengan atribut (data) dan metode (fungsi/perilaku).

Konsep utama: Kelas, Objek, Enkapsulasi, Abstraksi, Pewarisan, Polimorfisme.

MVC (Model-View-Controller)

Pola desain arsitektur perangkat lunak yang memisahkan aplikasi menjadi tiga komponen utama:

  • Model: Data dan logika bisnis
  • View: Tampilan antarmuka pengguna
  • Controller: Penghubung antara Model dan View

Teknologi yang Digunakan

HTML5 CSS3 JavaScript Bootstrap 5 Chart.js MySQL AJAX

Penutup

Kesimpulan, saran, dan ucapan terima kasih

Kesimpulan

  • Konsep pemrograman berorientasi objek (OOP), arsitektur MVC, dan penggunaan Git/GitHub berhasil diterapkan dalam pengembangan aplikasi web nyata dengan framework Laravel.
  • Modul Admin Panel (Dashboard, Manajemen Produk, Transaksi, dan Laporan) berhasil dikembangkan dengan fitur lengkap.
  • Teknologi industri (Laravel 12, MySQL, Bootstrap 5, Chart.js) dikuasai dengan baik.
  • Sistem keamanan dasar (Role-Based Access Control) berhasil diterapkan untuk membedakan hak akses antara admin dan kasir.
  • Kemampuan kolaborasi dan problem-solving terasah melalui pembagian tugas dalam tim dan penyelesaian masalah teknis.

Saran

Untuk Sekolah (SMK Negeri 1 Manggar):

  • Mempertimbangkan untuk memasukkan framework Laravel ke dalam mata pelajaran Pemrograman Web Lanjutan
  • Menambah jam pelajaran dan praktik kolaborasi menggunakan Git/GitHub
  • Menerapkan lebih banyak pembelajaran berbasis proyek nyata

Untuk Siswa SMK Negeri 1 Manggar:

  • Jangan hanya puas dengan dasar-dasar Laravel, eksplorasi fitur lanjutan
  • Mulai membangun portfolio proyek pribadi di GitHub sejak dini
  • Mempelajari React.js atau Vue.js sebagai pelengkap skill Laravel

Ucapan Terima Kasih

Saya mengucapkan terima kasih yang sebesar-besarnya kepada semua pihak yang telah mendukung dan membimbing selama pelaksanaan PKL, terutama kepada SMK Negeri 1 Manggar, PT Metanouva Informatika, pembimbing lapangan, rekan kerja, dan keluarga.
